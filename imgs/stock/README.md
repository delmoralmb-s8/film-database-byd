# Imágenes de stocks de película

Pon aquí las imágenes de los stocks. Acepta `.jpg` o `.png`.
Tamaño recomendado: cuadrado, mínimo 100×100px.

Si la imagen no existe, el app usa automáticamente un color chip de fallback.

## Para cualquier otro stock

El slug sigue la regla: `{marca}-{nombre}` en minúsculas, espacios y caracteres especiales → guión.

Ejemplos:
- `Kodak Pro Image 100`  →  `kodak-pro-image-100.jpg`
- `Cinestill BwXX`       →  `cinestill-bwxx.jpg`
- `Ilford Pan F Plus 50` →  `ilford-pan-f-plus-50.jpg`
